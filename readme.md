# Simple React starter

## Tech stack

*react
*redux
*react-router
*vite (with module federation)
*typescript
*css modules (for styles isolation)

## Installation

1. cd ./host
2. npm i
3. cd ./remote
4. npm i

## Running the project

1. cd ./host
2. npm run build
3. npm run preview

4. cd ./remote
5. npm run build
6. npm run preview

Running vite in preview mode and first building the projects ensures remoteEntry.js file is generated by remote
